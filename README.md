# csskit-less
一套移动端的样式拓展工具包

## 安装
```shell
npm i csskit-less --save
```

## 使用
```js
import 'csskit-less'
```
or 使用 `dist` 下的目录下 `csskit.css` or `csskit.min.css` 文件
